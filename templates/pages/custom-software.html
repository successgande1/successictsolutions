Some Software Features

  • Account App.
    • User Authentication: Admin Add and Manages Users with Roles.
    • Income & Expenses Input: Staff User can Add Daily Income and Expenses with dates.
    • Financial Summary: Income and Expenses are Summarise in User Account Dashboard on Daily Basis.
    • Financial Reports: Admin User can generate Financial Report using Date Ranges and exported to PDF.
  • Scholarship App.
    • Application: Candidates can Register and print Photo Card.
    • Scholarship Approval Admin Can Approve Candidates' scholarship Application and print out the list.
    • Application Summary: Summary of Submitted, Paid Processing Fee and Beneficiaries is displayed on Cards in Dashboard.
  • Daily Saving App.
    • Authentication: Admin Can Add Staff Users with different Roles.
    • Customers: Designated Staff User can Create Customer Saving Account with Unique Number.
    • Deposit & Withdrawal: Designated Staff User can add Customer Deposit, Withdrawal Request, and Withdrawals with Printable Slip.
    • Financial Reports: Designated Staff User can view and generate Reports using Date Ranges.
  • Event Mgt. App.
    • Authentication: Admin Can Add Staff Users with different Roles.
    • Events and Tickets: Admin User Can Manage Events and Tickets
    • Tickets & Pins: Admin User can can Generate PINs on Tickets.
    • Guest Register: Intending Event Guests can Activate Event Ticket Pin and Register with Printout Slip.
    • Reports: Admin User can generate Timely Reports.
